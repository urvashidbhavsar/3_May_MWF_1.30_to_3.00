loop is a count number of iteration

there is 6 type of loop 
(1) while loop 
(2) do while loop 
(3) for loop 
(4) for of loop  (array)
(5) for in loop  (object)
(6) foreach loop (array/object)

while loop : (syntax)
while(condition){
    statement;
    increment/ decrement
}

do..while loop : (syntax)
do{
    statement;
    increment/ decrement
}while(condition)

for loop :
initialization = declaration
for(initialization; condition; increment/decrement){
    statement
}
