css => cascading style sheet.

= used to make responsive website design.
= used to make perfect partitions.
= use to make animation on webpage.

there are 3 type of create css on webpage.
(1) inline    = using style attribute
(2) internal  = using <style> tag in <head> section.
(3) external  = create an external file with .css extension and link with <link> tag in html file.

syntax : 
(1) inline 
 style = "property : value;"

(2) internal / external 
selector{
    property : value;
}

selector ??
(1) element selector 
(2) id selector 
(3) class selector 
(4) group selector
(5) universal / global selector
(6) decendent selector
(7) attribute selector
